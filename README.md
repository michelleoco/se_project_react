# WTWR (What to Wear?)

## Description

A full-stack web application called “WTWR”. This application reads weather data from a Weather API and then recommends suitable clothing to the user based on that data.

## Tech Stack

- Figma for design content
- HTML
- CSS
- JavaScript
- React.js + Vite
- Node.js
- NPM
- APIs
